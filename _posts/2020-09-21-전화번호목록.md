### 전종원
```
def solution(phone_book):
    sorted_book = sorted(phone_book)
    print(sorted_book)
    for i in range(len(sorted_book)-1):
        for j in range(i+1, len(sorted_book)):
            if sorted_book[i] == sorted_book[j][:len(sorted_book[i])]:
                return False
    return True
```
